# tts-tools
This repository includes a set of tools I use for preparing datasets for Piper TTS (text-to-speech) voice training.

The tools are in varying states of roughness. Some bear the signs of being made in a hurry with little regard to usability for anyone but myself.

I do believe, though, that some of the tools might be useful for others who are interested in working with Piper TTS voice training. And as such I've decided to make them public.

Some tools are written in Python3. Others are written in C++.

## The `_incoming` folder
All tools in the `_incoming` folder are mostly in a rough state. That doesn't mean they don't work for whatever I was using them for. It just means that I've spend very little time on usability. For instance some will have hardcoded parameters. Please check each script or programs sources before using them.

I take zero responsibility if one of the tools messes something up for you! You have been warned!!! :)
